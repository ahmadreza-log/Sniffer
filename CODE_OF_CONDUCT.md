# Code of Conduct

## Our pledge

We are committed to providing a welcoming and harassment-free experience for everyone. We expect all contributors and users to be respectful and constructive.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, and issues or pull requests that do not align with this code of conduct. Serious or repeated violations may result in a ban.

Report unacceptable behavior by opening a private issue or contacting the maintainers directly.
